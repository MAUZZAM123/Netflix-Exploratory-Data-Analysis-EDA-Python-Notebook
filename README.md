# Netflix-Exploratory-Data-Analysis-EDA-Python-Notebook

Project Overview

This project performs an in-depth analysis of the Netflix dataset using Python in Jupyter Notebook. The objective is to explore content distribution, growth trends, and key insights related to Movies and TV Shows through data cleaning, exploratory data analysis (EDA), and visualizations.
The notebook demonstrates end-to-end data analysis from raw dataset to meaningful business insights.


Objectives

Analyze Movies vs TV Shows distribution
Identify yearly content growth trends
Examine country-wise content distribution
Analyze release year patterns
Explore duration trends
Generate business insights from data
